# 📈 TheStockie ★~(◠‿◕)

> *"To the moon — but like, with actual research this time."*

## What is this?

TheStockie is an advanced stock analysis tool powered by AI. No WSB memes, no diamond hands propaganda — just cold, hard fundamentals to help you make slightly less terrible financial decisions.

### Features

- **Stock Search & Quotes** — Find any ticker before you YOLO into it
- **Fundamentals Analysis** — Balance sheets, cash flows, EPS, and all the boring stuff that actually matters
- **AI-Powered Stock Analysis** — Let the robots do the DD so you don't have to
- **Stock Screener** — Filter thousands of stocks because apparently you have opinions
- **Earnings Calendar** — Know when to hold your breath (and your shares)
- **Analyst Ratings** — See what the suits on Wall Street think (then do the opposite, probably)
- **Insider Trading Tracker** — Legally watch what insiders are doing with their own money
- **Blog** — Actual written content that isn't a tweet thread

## Tech Stack

Built with the boring-but-reliable stuff that actually ships:

- **Next.js 15** + **React 19** — Because we live on the bleeding edge
- **tRPC** — Type-safe APIs, so no more `undefined is not a function` at runtime
- **Convex** — Database that doesn't make you write SQL at midnight
- **Tailwind CSS** + **shadcn/ui** — Pretty without trying too hard
- **Recharts** — For those satisfying green lines going up (we wish)
- **NextAuth 5** — Discord & Google login, because passwords are a 2010 problem

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) and start your due diligence journey.

> **Note:** You'll need a [Financial Modeling Prep](https://financialmodelingprep.com/) API key. Set it in your `.env`. Without it, the app is just a very pretty loading spinner.
